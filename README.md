# STM32 Project

Цей проєкт реалізує дистанційно керований автомобіль на базі мікроконтролера STM32F407. Автомобіль приводиться в рух чотирма DC-моторами, керованими за допомогою драйвера L293D. Система отримує команди бездротово через приймач ELRS Nano 2.4GHz, підключений до пульта Radiomaster TX12.

У коді ініціалізуються необхідні периферійні модулі, включаючи GPIO, таймери та USART. Виводи GPIO порту D налаштовані як виходи для керування моторами, а порти A і B використовуються для альтернативних функцій — генерації PWM сигналів та прийому даних від приймача. Таймери генерують PWM-сигнали для регулювання швидкості кожного мотора, а модуль USART відповідає за прийом команд від приймача.

Автомобіль використовує протокол CRSF для отримання даних від пульта керування. Значення каналів видобуваються з отриманих пакетів. На основі прийнятих команд програма визначає напрямок руху автомобіля та встановлює відповідні виходи для моторів.

Система працює від акумулятора, встановленого на акриловій платформі з колесами.

Основне обладнання та компоненти що використовувалися:

Мікроконтролер STM32F407

Драйвер L293D DC Motor Driver

4 DC моторa

Приймач ELRS Nano 2.4GHz

Пульт керування Radiomaster TX12

Логічний аналізатор

Акумулятор для живлення системи

Акрилова платформа з колесами
